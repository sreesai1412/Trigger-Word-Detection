# Trigger-Word-Detection
In this notebook, a speech dataset is constructed and an algorithm for trigger word detection (sometimes also called keyword detection, or wakeword detection) is implemented. Trigger word detection is the technology that allows devices like Amazon Alexa, Google Home, Apple Siri, and Baidu DuerOS to wake up upon hearing a certain word.
